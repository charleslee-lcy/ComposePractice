云报全媒体绩效考核客户端项目架构总览
==============================

# app

主壳工程，整合APP正常运行所需要的模块，包含UI框架和应用级导航。

# core

核心通用库模块，可以依赖于其它的core。

- common 公共资源，工具类等
- data 数据层相关，数据模型等
- network 网络层相关
- widget UI组件，公共轮子

# feature

核心功能层模块，feature彼此之间隔离，可以依赖底层的core。

- basis 基础功能模块，如登录、注册、我的等页面功能
- review-data 绩效考核数据模块
- review-manager 绩效考核管理模块
